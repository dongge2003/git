### git使用
**命令行**  

```
git version     //查看git版本
git config --global user.name "****"    //
git config --global user.email "****"   //
git init        //初始化git
git add ***     //将文件加入暂存区
git commit      //提交,之后会进入vim编辑器编辑调教说明
git commit -m "****"    //提交简化版本
git log     //查看历史提交
git reset --hard "***********************"  //回退版本
git branch ***      //创建分支
git branch -a       //查看所有分支
git checkout ***    //切换分支
git merge ****      //合并分支       


```
